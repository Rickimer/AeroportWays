Проект-пример асинхронного многопоточного API с использованием стороннего api
Вычисляет расстояние между двумя аэропортами на основании 2 iata-кодов аэропортов

Пример вызова стороннего API
https://places-dev.cteleport.com/airports/AMS

При запросе Post api немедленно возвращает id для последующего запроса результата и начинает процедуру асинхронных вычислений
